# qwen_vl_test

Скрипт для автоматического описания изображений с помощью ИИ-модели.

## О выборе модели

Изначально задача стояла в запуске более крупной 7B-версии модели для максимальной детализации. Однако из-за высоких требований к вычислительным ресурсам (для 7B моделей требуется мощный GPU с большим объемом памяти) был выбран оптимальный вариант — модель **Qwen2.5-VL-3B-Instruct**.

Результаты с описаниями для каждого изображения сохраняются в файл `image_descriptions.json`.
